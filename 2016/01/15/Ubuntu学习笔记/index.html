解决SSH连接慢的问题

  • 调试办法：
    ssh -v host 查看调试信息
    time ssh root@host exit
  • 解决办法
  1. 关闭DNS反向解析
    vi /etc/ssh/sshd_config 增加或修改 UseDNS=no
  2. 关闭SERVER上的GSS认证
    vi /etc/ssh/sshd_config 修改 GSSAPIAuthentication no
  3. 修改resolv.conf文件
    vi /etc/resolv.conf 删除所有不使用的IP
  4. 修改hosts文件
    vi /etc/hosts文件中把客户端的IP和HOSTNAME加入
  5. 打开SERVER上的IgnoreRhosts参数
    vi /etc/ssh/sshd_config 修改IgnoreRhosts yes
  6. 重启SSH服务
    service sshd restart 或 /etc/init.d/ssh restart
